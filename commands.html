- Commands -

Permission Command Usage
basicx.info /bxinfo View the author/version page.
basicx.admin.cfgtest /configtest View the config test message.
basicx.gamemode.survival /gms Set gamemode to survival.
basicx.gamemode.creative /gmc Set gamemode to creative.
basicx.gamemode.spectator /gmsp Set gamemode to spectator.
basicx.gamemode.adventure /gma Set gamemode to adventure.
basicx.gamemode.others /gm (player) Modify another players gamemode.
basicx.gamemode.* /gm Access to all gamemode permissions.
basicx.tp /tp (player) Teleport to a player.
basicx.tp.others /tp (player) (player) Teleports a player to another player.
basicx.tpall /tpall Teleports all players to you.
basicx.tpall.others /tpall (player) Teleports all players to a specified player.
basicx.tpa /tpa (player) Request to teleport to a player.
basicx.tpaccept /tpaccept Accept a teleport request.
basicx.tpdeny /tpdeny Deny a teleport request.
basicx.tpatoggle /tpatoggle Toggle incoming teleport requests.
basicx.fly /fly Enable/Disable flight.
basicx.fly.others /fly (player) Enable/Disable flight for other players.
basicx.flyspeed /flyspeed (number) Set flight speed.
basicx.walkspeed /walkspeed (number) Set walking speed.
basicx.feed /feed Restore your hunger.
basicx.god.others /god (player) Enable/Disable god mode for another player.
basicx.god /god Enable/Disable god mode.
basicx.feed.others /feed (player) Restore the hunger of other players.
basicx.heal /heal Restore your health.
basicx.heal.others /heal (player) Restore the health of another player.
basicx.setlevel /setlevel (player) (amount) Change the level of a player.
basicx.give64 /give64 (itemname) Give yourself a stack of items.
basicx.more /more Get more out of what you're holding.
basicx.admin.stats /stats (player) View the stats of a player.
basicx.getpos /getpos (player) Get the position of a player.
basicx.time.day /day Set server time to day.
basicx.time.night /night Set the time to night.
basicx.time /time Get the server/real time.
basicx.settime /time set (tick) Set the servers time in ticks.
basicx.ptime /ptime (tick) | /ptime reset Set your custom time.
basicx.nick /nick | /nick reset Change your nickname.
basicx.nick.color /nick (name w/ colors) Add colors to your nickname.
basicx.msg /msg (player) (message) Send a message to a player.
basicx.msg.chatcolor /msg (player) (message with color) Message a player with color.
basicx.repair /repair Repair the item in hand.
basicx.repair.enchanted /repair Allows you to repair an enchanted item.
basicx.ignore /ignore Ignore a player.
basicx.unignore /unignore Stop ignoring an ignored player.
basicx.ignore.bypass /ignore Allows the exemption of being ignored.
basicx.ignore.all /ignore Allows you to ignore all players regardless of permissions/op.
basicx.rules /rules Allow players to view rules specified in the config file.
basicx.setspawn /setspawn Set a world spawn.
basicx.spawn /spawn Teleport to spawn.
basicx.jump /jump Allows you to teleport to the block they are looking at.
basicx.tppos /tppos [X] [Y] [Z] Teleport to a specified coordinate.
permbasicx.teleportcooldown.bypass /tpa, /spawn Allows you to bypass a teleport cooldown.
basicx.ip /ip View the server's IP.
basicx.chatcolor Allows color formatting in chat.
basicx.staffchat @ [message] Allows you to send and receive staff chat messages.
basicx.motd /motd View the server's MOTD.
basicx.extinguish /ext Remove a fire
basicx.extinguish.others /ext [player] Allows you to extinguish other players.
basicx.fire /fire [player] [seconds] Set fire to a player for a number of seconds.
basicx.nightvision /nv Toggle nightvision.
basicx.nightvision.others /nv [player] Toggle nightvision for others.
basicx.clearchat /cc Clear the global chat.
basicx.personalclearchat /pcc Clears your personal chat.
basicx.healcooldown.bypass /heal Allows you to bypass the /heal cooldown.
basicx.feedcooldown.bypass /feed Allows you to bypass the /feed cooldown.
basicx.commandcooldown.bypass /[command] Allows you to bypass the command cooldown.
basicx.fakejoin /fakejoin Sends your fake join message to the server.
basicx.fakeleave /fakeleave Sends your fake leave message to the server.
basicx.sethome /sethome [name] Set a home.
basicx.home /home [name] Teleport to specified home.
basicx.delhome /delhome [name] Delete an existing home
basicx.sethome.(number) /sethome [name] Allows you to grant a player a specific number of homes.
basicx.commandcooldown.bypass /[command] Allows you to bypass the command cooldown.
basicx.homes /homes View a list of your homes.
basicx.worldtp /worldtp [world] Teleport to a specific world.
basicx.setwarp /setwarp [name] Set a warp.
basicx.warps /warps View a list of warps you have permission to travel to.
basicx.warp /warp [name] Teleport to a warp.
basicx.warp.(name) /warp [name] Allows to set specific warp permissions.
basicx.staff /staff Opens a customizable staff GUI.
